<h1>Laboratório SIEM</h1>



<h2>Descrição</h2>
Configurei um SOC doméstico básico no Azure do zero. Usando uma assinatura gratuita do Azure, criei uma máquina virtual (VM), abri a máquina na internet como um honeypot e encaminhei os logs para um repositório central. Em seguida, integrei o Microsoft Sentinel para analisar os dados de ataques reais.
<br />


<h2>Ferramentas e Serviços usadas</h2>

- <b>Azure</b> 
- <b>Máquina Virtual</b>
- <b>Microsoft Sentinel</b>
- <b>Log Analytcs Workspace</b>

<h2>Ambiente Usado </h2>

- <b>Windows 10</b> (21H2)
